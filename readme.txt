"make" will compile the program.
"java Breakout" will execute the program.

Additional Features:

1.
Press "p" to pause the game;
Press "r" to restart the game.

2.
Each player has 3 lives. If you used up all the 3 lives, then the game is over.

When the game is over, the player can enter his/her name.

The players' scores and names will be recorded in a txt file named "highscore".

There are most 10 records in the file, and the scores are in the decreasing order.
